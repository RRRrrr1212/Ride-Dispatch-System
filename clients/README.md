# JavaFX Clients

此目錄將包含三個 JavaFX 前端應用程式：

## 目錄結構 (規劃中)

```
clients/
├── passenger-app/     # 乘客端
├── driver-app/        # 司機端  
└── admin-console/     # 管理後台
```

## 技術堆疊

- JavaFX 17+
- FXML (UI 佈局)
- CSS (樣式)

## 開發順序

1. 先完成後端 API 
2. 三個視窗可併行開發
3. 使用 Polling 機制同步狀態

## 視窗規格

模擬手機螢幕尺寸：
- 寬度: 375px
- 高度: 667px
