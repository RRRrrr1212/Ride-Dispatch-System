## Rider (乘客端) - 5 個畫面

| # | 畫面 | 路由 | 功能 | 優先級 |
|---|------|------|------|--------|
| 1 | **首頁** | `/rider/home` | 「你要去哪裡？」入口 | P0 |
| 2 | **叫車** | `/rider/request` | 選點 + 車種 + 預估 + 確認 | P0 |
| 3 | **等候** | `/rider/waiting/:id` | 等待司機接單 (Polling) | P0 |
| 4 | **行程中** | `/rider/trip/:id` | 顯示司機位置、進度 | P1 |
| 5 | **完成** | `/rider/completed/:id` | 費用明細 | P1 |

### 畫面流程

```
首頁 → 叫車 → 等候 → 行程中 → 完成
  │                      ↓
  └── 取消 ←────────────┘
```

## Driver (司機端) - 4 個畫面

| # | 畫面 | 路由 | 功能 | 優先級 |
|---|------|------|------|--------|
| 1 | **待命** | `/driver/dashboard` | 上線/下線 + 可接訂單列表 | P0 |
| 2 | **訂單詳情** | `/driver/order/:id` | 乘客資訊 + 接單/拒絕 | P0 |
| 3 | **行程中** | `/driver/trip/:id` | 開始/完成行程 | P0 |
| 4 | **歷史** | `/driver/history` | 完成的行程列表 | P2 |

### 畫面流程

```
待命 (上線) → 點擊訂單 → 訂單詳情 → 接單 → 行程中 → 完成 → 回到待命
                          ↓
                        拒絕/忽略 → 回到待命
```

## 元件清單

### 共用元件

| 元件 | 用途 |
|------|------|
| `StatusChip` | 訂單狀態標籤 (等待中/進行中/已完成/已取消) |
| `OrderCard` | 訂單卡片 (顯示路線、金額) |
| `MapView` | 簡易地圖 (顯示座標點) |
| `LoadingSpinner` | 載入動畫 |
| `VehicleSelector` | 車種選擇器 |

### 乘客端專用

| 元件 | 用途 |
|------|------|
| `LocationInput` | 地點輸入框 |
| `FareEstimate` | 車資預估顯示 |
| `DriverInfoCard` | 司機資訊卡片 |

### 司機端專用

| 元件 | 用途 |
|------|------|
| `OnlineToggle` | 上線/下線切換 |
| `OfferList` | 可接訂單列表 |
| `TripControls` | 行程控制按鈕 (開始/完成) |
